# Project Agent Instructions

This repository delegates its investment-research operating protocol to the upstream Mira submodule at `Mira/`.

Before substantive work:

1. Ensure the submodule is present with `git submodule update --init --recursive`.
2. Read and follow `Mira/AGENTS.md` as the governing agent rules.
3. Load `Mira/OPERATING_CONTRACT.md` and `Mira/AGENT_QUICKSTART.md` before routing formal research.
4. Use `Mira/START_HERE.md` for onboarding and task examples.
5. Keep generated cases, reports, evidence logs, and other research artifacts inside the locations prescribed by Mira.

The upstream Mira rules take precedence over this wrapper when they conflict. Do not treat Mira as an investment adviser, trade executor, signal service, or autonomous portfolio manager.
